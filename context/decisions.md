# Decision Log

**Append-only.** Add new decisions at the bottom. Never edit or delete an entry. If a decision turns out wrong, write a new entry that supersedes it and name the one it replaces. This file is the record of why things are the way they are — any agent can trust it without asking around.

## Template

Copy this block to the bottom for each new decision:

```
### YYYY-MM-DD — <short title>

- **Made by:** <role — Planner / Manager / Coder / Reviewer / Human>
- **Context:** <what prompted this — the situation and options considered>
- **Decision:** <what we decided, stated plainly>
- **Impact:** <what this affects going forward; note if it supersedes an earlier decision>
```
